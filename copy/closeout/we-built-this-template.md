# Closeout "We Built This" Marketing-Page Template

*Multi-lane MLP copy · 2026-05-28 · Status: DRAFT — pending legal review (see flags)*

## Design constraints
- Light backgrounds globally — white #FFFFFF page, warm-white #FAFAF8 section cards, cream #FDF8F0 accent stripes
- Archivo body, Archivo Black for the project name hero and section heads
- Killer App red #E8443A for the primary "Build with [GC Company]" CTA
- Hero photo full-bleed at top (Owner-approved selection)
- Stage timeline rendered as a horizontal pill row: Size Up · Lock · Plan · Build · Adapt · Collect · Reflect (locked names — never substitute)
- Cast credits with avatar + role + link to each Lane member's BKG profile
- Owner reflection and GC reflection rendered as pull-quote blocks
- Sharable via direct URL, social-card preview, and one-tap-to-share buttons (no popups)

## ⚠ Legal review flags
- **"Every dollar accounted for"** — sounds warranty-adjacent. Soften, qualify, or remove before any paid build publishes a page.
- **"Every photo documented"** — UGC publication rights. Each Lane that contributed photos/sketches/videos likely needs publication consent captured at the moment of upload (not buried in onboarding T&Cs).
- **Owner reflection + GC reflection blocks** — user-generated content. Need UGC terms covering defamation indemnity, right to remove, publication rights.
- **Cast credits with link-backs** — each Lane member's profile is publicly attributed to this build. Confirm profile-attribution requires no separate per-project consent beyond platform terms.
- **Codes met section** — claims of code compliance carry liability if inaccurate. Either tie strictly to inspector pass records OR add a disclaimer.
